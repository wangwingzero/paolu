客户端配置

参数就这几个，别在 UI 里瞎猜。机器已经够烦了。

通用参数

下载地址https://github.com/farion1231/cc-switch
Base URLhttps://paolu.hudawang.cn/v1
API Key你自己的 key

cc-switch 里只填这两个

URL = https://paolu.hudawang.cn/v1
API Key = YOUR_API_KEY

别额外发明配置项。你需要的是能用，不是参加参数设计大赛。

最短流程

  1. 下载 cc-switch
  2. 新建自定义配置
  3. 填 URL
  4. 填 key
  5. 重新打开 Codex

手搓 curl 测试

curl https://paolu.hudawang.cn/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"

这是备用确认手段，不是主线剧情。
