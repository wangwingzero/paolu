客户端配置

参数就这几个，别在 UI 里瞎猜。机器已经够烦了。

通用参数

Base URLhttps://paolu.hudawang.cn/v1
API Key你自己的 key
模型按后台可用模型选择

Codex / OpenAI Compatible

base_url = "https://paolu.hudawang.cn/v1"
api_key = "YOUR_API_KEY"

最容易出错的是把地址写成根域名却没带 /v1。

Cherry Studio

  • 供应商类型选 OpenAI 兼容。
  • API 地址填 https://paolu.hudawang.cn/v1
  • 密钥填你自己的 token。

Open WebUI / 其他兼容客户端

只要支持 OpenAI Compatible，大概率都能接。前提是你别把斜杠和域名抄出灵异事故。
