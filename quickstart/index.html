Last Chance Guide

快速开始

真的不复杂。复杂的是人类喜欢把简单事情搞得像召唤邪神。

你需要准备什么

  • 一个能登录后台的账号。
  • 一枚你自己创建的 API Key。
  • 这个地址：https://paolu.hudawang.cn/v1

三步接入

第一步：创建令牌

登录后台后创建一个 token。名字随意，但别起成 test123 然后两天后问自己这玩意是谁的。

第二步：填客户端

把 https://paolu.hudawang.cn/v1 填到 Base URL，把你的 key 填到 API Key。

第三步：先测，再幻想

先确认最小请求能过，再去改模型、代理、系统提示词和那些带来幻觉的高级选项。

最小测试

curl https://paolu.hudawang.cn/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"

能拿到模型列表，说明门是开着的。拿不到，就去 FAQ 对照死因。
