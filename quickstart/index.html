Last Chance Guide

快速开始

真的不复杂。复杂的是总有人喜欢把“填 URL 和 key”演成一场软件祭祀。

你需要准备什么

  • cc-switch
  • 这个地址：https://paolu.hudawang.cn/v1
  • 你自己的 API Key。

三步接入

第一步：下载 cc-switch

先去 GitHub 下载。兵器都没领，别急着问剑招。

第二步：新建自定义配置

在 cc-switch 里新建一个自定义配置，把 https://paolu.hudawang.cn/v1 填到 URL。

第三步：填 key 然后打开 Codex

把 API Key 填进去，保存，重新打开 Codex。到这里就够了，别再给流程加支线任务。

顺手给你放三个入口

https://paolu.hudawang.cn/

主站入口。大多数人真正要进去配置和使用的就是这里。

https://luo.hudawang.cn/

另一套站点入口。需要的时候直接点，不需要的时候也别把它和主站搞混。

https://status.hudawang.cn/

状态页。怀疑服务抽风之前，先来这里看看到底是谁先跑路了。

如果你非要测一下

curl https://paolu.hudawang.cn/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"

能拿到模型列表，说明路没堵死。拿不到，先检查是不是 URL 或 key 抄错了。
