Do Not Mix Them Up

先分清：`paolu` 和 `luo` 不是同一个网站

很多人翻车就翻在这里：两个站入口不一样，API URL 也不一样。你用哪个站，就填哪个站对应的 /v1。

如果你用 `paolu` 站

站点入口：https://paolu.hudawang.cn/

https://paolu.hudawang.cn/v1

如果你用 `luo` 站

站点入口：https://luo.hudawang.cn/

https://luo.hudawang.cn/v1

API Key 示例

key 的格式就长这样。别把示例里的 X 当成真的一并抄进去。

sk-XXXXXXXXXXXXXXXXXXXXXXXX

The Bare Minimum

你真正需要抄的就这两样

  • 先下载：https://github.com/farion1231/cc-switch
  • API 地址：填你选中的站点对应 URL，不要混填。
  • API Key：格式示例是 sk-XXXXXXXXXXXXXXXXXXXXXXXX

Recommended Ritual

标准接入流程

  1. 下载并打开 cc-switch。
  2. 新建一个自定义配置。
  3. 把 API URL 填成你选中的站点对应 /v1。
  4. 把 API Key 填成你自己的 sk-... key。
  5. 保存后重新打开 Codex，就这么多，别自己给自己加戏。

cc-switch 里大概就填成这样
