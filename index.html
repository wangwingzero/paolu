Recommended Ritual

标准接入流程

  1. 下载并打开 cc-switch。
  2. 新建一个自定义配置。
  3. 把 Base URL 填成 https://paolu.hudawang.cn/v1。
  4. 把 API Key 填成你自己的 key。
  5. 保存后重新打开 Codex，就这么多，别自己给自己加戏。
curl https://paolu.hudawang.cn/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"

如果你非想确认通不通，就跑这条。否则照着填完直接用。
