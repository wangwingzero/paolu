标准接入流程

  1. 登录后台，创建令牌。
  2. 在客户端填入 Base URL 和 API Key。
  3. 先测一个最简单的请求，别上来就一顿神奇配置。
  4. 如果报错，先看 常见问题，别先怀疑地球磁场。
curl https://paolu.hudawang.cn/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
