Hall Of Failures

常见问题

程序报错的时候像谜语人。这里把常见死法翻译成人话。

401 / Unauthorized

通常是 key 不对、key 失效，或者你把别人的 key 当成自己的命根子。

模型列表拿不到

优先检查 Base URL 是否是 https://paolu.hudawang.cn/v1。少一个 /v1，世界观都不一样。

显示没额度

说明账户、令牌或额度状态有问题。先看后台，不要第一反应是“是不是宇宙在针对我”。

客户端能保存，但请求一直报错

先用 curl 测。GUI 很会装，命令行一般比较诚实。

国内访问慢

这也是为什么文档站单独放在 Cloudflare Pages。至少让你查文档的时候别先经历一次修行。
